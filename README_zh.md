# PhiTogether

[English](README.md) | 中文

[iOS 应用](https://testflight.apple.com/join/jMbjenat)

[服务状态](https://status.phitogether.fun)

将多人游戏与Phigros玩法结合起来！一个基于Phigros玩法的非盈利性开源社区游戏。集成PhiZone。
